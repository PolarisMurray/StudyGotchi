# Live2D Models Directory

Place your Live2D model files here. Each model should be in its own directory with the following structure:

```
model-name/
  ├── model.json (or model.moc)
  ├── textures/
  │   ├── texture_00.png
  │   └── ...
  ├── motions/
  │   └── ...
  └── physics.json (optional)
```

You can find free Live2D models at:
- https://github.com/zenghongtu/live2d-model-assets
- https://www.live2d.com/en/download/sample-data/

Make sure to respect the licensing terms of any models you use.
